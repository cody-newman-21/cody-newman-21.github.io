CSS Concepts

What is the difference between Margin, Border, and Padding?

As part of Sprint 2 (week 2) while studying web develeopment with Dev Academy, I have been tasked with creating this webpage and blog to showcase the skills I've learned using HTML and CSS. In particular, we're to explain a concept we have learned as if we were teaching somebody for the first time. The concepts I've chosen to explain are margin, border and padding, as I feel explaining them will help further my own understanding of them, while putting them to use behind the scenes on this page.

CSS (Cascading Style Sheet) is a language used to determine the design of a HTML webpage and its individual sections, known as 'elements', including their shapes, colours, positioning on the page and more. All elements can be thought of as a 'box', which can be defined using the margin, border, padding and the actual content being displayed. Collectively, they're known as the 'box model', although they can each be used alone and irrespective of each other.

Margin

Margins are the outermost part of the box model. They're transparent and used to create space around and between elements. With CSS, you have complete control of margins. You can define the top, right, bottom and left margins based on where you want your element to sit on the page, using units such as px (pixels) or %. Border

A border is a little more self explanatory. It is a border/outline that goes around the content of the element, such as the border around this paragraph. Using CSS, you can specify the border including its colour, width, (using units such as 'px' or specifying 'thin'/'thick' etc.) and style, (for example: Solid, dotted or dashed). You can define each side (top, right, bottom and left) of a border, this is useful if you only want to have a border on one or more sides, or would like to style one or more sides indiviually. Padding

Padding is the CSS property used to create space around an element's content, inside of any defined borders. Generally, the padding is transparent but unlike the margin property, it can be coloured if you wish. With CSS you can specify the padding on each individual side of the element (top, right, bottom and left). This is also achieved by using units such as 'px' (pixels) or %.
